本机配置

邮箱连接

仅支持 TLS。建议使用应用专用密码或授权码，而非邮箱主密码

保存到 imap.mailbox；空数组表示全部 mailbox
保存后即可抓取

过滤条件

用于邮箱搜索。多个关键词会一起匹配；主题与正文可独立开启

如果填写了起止日期，将优先使用这个范围。留空则使用下面的最近天数

输出目录

先保存所有发票或行程单原件，再识别和整理。这样即使识别失败，也不会丢文件

命名规则

先用不会冲突的文件名保存。识别成功后，可以按开票方、金额、日期等信息自动改名

示例：阿里云-3418.20.pdf
示例：2026-05-20-a91f0e2c.pdf
实时预览
国家电网-318.42.pdf

识别服务

用于从已归档发票或行程单解析销售方、金额、票号、日期和类型。未启用时原文件仍完整保留

这些鉴权信息只写入本机配置；运行 efapiao 时会作为本地环境变量透传，不会写入发票文件。

智能查找兜底

当普通规则找不到发票时，可让智能查找根据邮件正文和链接判断下一步。但默认不启用

网页自动下载

用于第三方开票站点。首次使用前请运行 npx playwright install chromium

网络重试

第三方站点或直链下载临时失败时自动重试；仍失败会进入待人工队列并在运行结束列出

保存到 network.retries 与 network.retryDelayMs
修改后自动保存

开发者重置

删除本机缓存邮件、已归档发票、待确认队列、识别结果和运行记录。配置不会删除。
