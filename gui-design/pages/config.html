§1 · IMAP 连接 imap
仅支持 TLS。建议使用应用专用密码或授权码，而非邮箱主密码。
§2 · 过滤条件 filter
用于 IMAP SEARCH。多个关键词以 OR 组合；主题与正文可独立开启。
SINCE N days · 0 表示全部
§3 · 输出目录 output
已归档的 PDF、未识别的 .eml、清册 CSV 各自的落地路径。支持相对或绝对路径。
§4 · 命名规则 rename
支持占位符 {seller} {amount} {date} {invoiceNo} {messageId}。字段缺失会自动回退到 fallback 模板。

§5 · OCR 提供商 ocr
用于从 PDF 解析销售方、金额、票号、日期。未启用时所有发票走 fallback 命名。
§6 · LLM 兜底 llm
仅当四个标准提取器都未命中时启用。把正文 + 链接列表交给 LLM 判断 "直链 / 跳转 / 都不是"。
§7 · Playwright playwright
用于第三方开票站点。首次使用前请运行 npx playwright install chromium。
